## 面试
- 面试主要是考 java基础 + java项目 + 计算机网络 + 计算机系统 + 数据库（直接写sql很少考）

- 时间紧计网和系统看我下面提到的小林coding的几篇，然后看一下代码随想录八股文速记版常考的计算机基础的总结即可

- 面试前快速掌握八股：[代码随想录八股文速记版](代码随想录知识星球-八股文速记版_unlocked.pdf)（一共38面，网络17问，操作系统12问，数据库12问）
